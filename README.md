# Information retrieval

Prerequisites
- Python 3.8+
- uv

Installation:

```bash
git clone git@github.com:artyomrabosh/information_retrieval.git
cd information_retrieval
uv sync
```

Running:
1) Download dataset from notebooks/load_dataset.ipynb or load your own
2) Run app.py
```bash
uv run web/app.py
```
